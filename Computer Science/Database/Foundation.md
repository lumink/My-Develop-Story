## 01. Database Definition
- 구조화된 데이터의 집합으로 데이터에 쉽게 접근하고 관리하기 위해 사용된다.
- file system, cloud storage 등 컴퓨터 시스템에 저장되며 데이터베이스관리시스템(DBMS)에 의해 관리되고 저장된다.
